⚡ Fast Picks — exam shortcuts
Read the options first, kill the wrong numbers, then settle the flip. Each rule is cue → rule → trap.
Power in an oblique meridian — cue: a sphero-cyl Rx plus a named meridian.
  • ›Meridian = the cyl axis → answer is the sphere alone (sin²0 = 0). Meridian ⟂ to axis → sphere + cyl.
  • ✗Trap: any option that applies the *full* cylinder to an in-between meridian.
Resultant prism — cue: both a vertical and a horizontal prism are given.
  • ›r is the hypotenuse: always bigger than either component, smaller than their sum (v + h). A 45° split gives r ≈ 1.41× a component.
  • ✗Trap: the option equal to v + h (added instead of √), and anything smaller than the larger component.
Base direction — cue: lens sign + the wording.
  • ›"Decentered" → + same, − opposite. "From the OC" or OC ≠ PD → + opposite, − same.
  • ✗Trap: the twin option with your magnitude but flipped BI/BO — that is the keyed bait.
Prentice's rule — cue: a decentration (mm) and a lens power.
  • ›P = (mm × D) ÷ 10. Only one option's number matches; pick the base with the sign rule above.
  • ✗Trap: the option that forgot to ÷10 (used mm as cm), off by 10×.
Transposition — cue: "express in plus/minus cyl," or options with a flipped axis.
  • ›New sphere = sph + cyl; flip the cyl sign; axis ± 90 (keep it 0–180).
  • ✗Trap: an option whose sphere ≠ (old sph + cyl), or whose axis is not exactly 90° away.

D1Physical Optics

Light theories "NHMP": Newton (corpuscular) → Huygens (wave) → Maxwell (electromagnetic) → Planck (quantum). Huygens correctly predicted light slows in dense media.
Vergence signs: diverging = negative, converging = positive. Higher n → slower light and lower critical angle (diamond ~24.44°). Beyond the critical angle = total internal reflection.
Abbe ↔ index are inverse: higher index → lower Abbe → more chromatic aberration. UV-B = photokeratitis (welder's flash); UV-A → cataract. Thermal damage = permanent; photochemical = reversible.
Flat transposition: new Sph = S + C, new Cyl = −C, new Axis = ±90°.
Trap: higher index = higher Abbe = less aberration. No — inverse: higher index = lower Abbe = more chromatic aberration.
Trap: keratometry uses the true corneal index 1.376. No — it uses the keratometric index 1.3375.
Trap: Newton's theory predicted light slows in glass. No — Newton predicted speed-up; Huygens was right.

D1Geometric Optics

Always virtual, upright, minified: the convex mirror and the concave (minus) lens, at any object distance. Plane mirror = virtual, upright, same size, equal distance behind.
Sign rules: q positive = real, q negative = virtual; m positive = upright, m negative = inverted. Plus lens with object inside F = virtual, upright, magnified (a magnifier).
Prism: light bends toward the base, image displaces toward the apex. Esotropia → base out; exotropia → base in. 1° ≈ 1.75Δ.
Surface power needs BOTH the radius and the index difference (matched media = zero power). Apparent depth = actual ÷ n (shallowing). Lens movement: plus = against, minus = with.
Trap: a flat glass slab (plane refractor) adds power. No — it only displaces the ray laterally; exit ray stays parallel.
Trap: exotropia is corrected base out. No — exotropia → base in; esotropia → base out.
Trap: a convex mirror can form a real image. No — convex mirrors are always virtual, upright, minified.

D3Contact Lenses

Vertex correction matters above ±4.00 D. D_CL = D_spec/(1 − 0.012·D_spec). High minus → less minus as a CL; high plus → more plus. Low cyl (<1.00) → spherical equivalent (sph + ½ cyl).
Fit: flat = excess movement → mechanical trauma; steep = too little movement → hypoxia/edema. Soft lens diameter ≈ HVID (11.7 mm) + 2 mm. Toric prism ballast ~1–1.5Δ base down.
Materials/history: PMMA = non-gas-permeable (Touhy 1947). Soft HEMA = Wichterle (FDA 1971). Silicone hydrogel carries O₂ through the silicone matrix, not water.
Complications: microbial keratitis (infiltrate + staining + pain) is the emergency — Pseudomonas most common; Acanthamoeba with water exposure/extended wear. Polymegethism = irreversible hypoxia marker. GPC = deposits + upper-tarsus friction. BAK is toxic with soft lenses.
Monovision: dominant eye = distance. Ortho-K: temporary corneal reshaping + myopia control, up to ~−6.00 D / −2.00 D.
Trap: a −6.50 spectacle becomes −6.50 as a CL. No — vertex correction makes it ~−6.00 (less minus).
Trap: PMMA is gas permeable. No — PMMA is non-permeable; RGPs use different polymers.
